Data API

API 与数据接口

为团队、研究员和交易工具提供钱包调查、巨鲸雷达、交易员画像、钱包监控和每日情报的数据接口。

当前状态

API 文档为开放计划说明。正式接口会在数据源、权限、速率限制和安全审计完成后开放。

Endpoints

计划开放的接口

接口设计围绕“可解释情报”，不是只返回原始交易列表。

GET

/wallet/{address}

返回钱包基础画像、风险等级、行为标签和监控建议。

GET

/whales/flows

返回巨鲸买入、卖出、交易所流入流出和大额资金路径。

GET

/traders/{id}

返回交易员胜率、收益、回撤、公开钱包和跟踪价值。

POST

/watch/rules

创建钱包监控规则，例如转入交易所、大额转账或风险合约交互。

GET

/intelligence/daily

返回每日 5 条核心情报，覆盖巨鲸、风险、板块、聪明资金和宏观。

POST

/mbti/report

根据钱包或交易员行为生成 Crypto MBTI 人格画像。

安全原则

  • API 不要求用户私钥、助记词或交易签名。
  • 团队访问需要密钥、速率限制和权限隔离。
  • 敏感数据只返回必要字段，避免扩大隐私风险。

适用场景

  • 研究团队批量调查钱包。
  • 社群或 KOL 生成每日情报。
  • 交易工具接入钱包监控和巨鲸提醒。
