Investigation Playbooks

调查手册

把链上情报拆成可执行的工作流：发现线索、复核证据、沉淀报告、设置预警。适合新用户快速理解，也适合团队统一调查标准。

使用边界

调查手册只用于信息整理和风险复核，不构成投资建议、跟单建议或交易指令。

Wallet 基础手册

陌生钱包快速调查

当你拿到一个地址时，先判断链、余额、交易频率、实体标签和风险信号，再决定是否继续跟踪。

  1. 输入地址，确认链类型和地址完整性。
  2. 查看实体标签、交易所流入和异常授权。
  3. 把高价值线索写入报告或加入 Wallet Watch。

Whale 资金流

巨鲸建仓复核

大额买入不等于确定机会，需要看是否分批、是否多地址同步、是否与资产雷达和板块资金流一致。

  1. 查看 Top Whale Buys / Sells。
  2. 复核地址是否属于同一实体或关联集群。
  3. 连接资产雷达和报告库，记录后续观察点。

Trader 质量复核

交易员可信度调查

不要只看收益截图。稳定盈利、最大回撤、仓位节奏和公开观点与链上行为是否一致都要复核。

  1. 查看胜率、收益和回撤是否跨周期稳定。
  2. 复核公开观点与链上交易是否一致。
  3. 把高风险跟单对象标记为观察，而不是直接跟随。

Risk 高优先级

异常授权与风险合约响应

当钱包出现异常授权或高风险合约交互时，优先确认资产暴露范围，而不是急着解释价格方向。

  1. 记录合约地址、授权额度和交互时间。
  2. 查看后续是否发生资产转出或关联地址扩散。
  3. 进入预警中心，持续观察二次交互。

Token 资产观察

资产与板块观察

资产雷达适合回答“这个 Token 或板块现在值得关注吗”，但必须结合资金流、事件窗口和风险信号。

  1. 先按主流币、AI、DeFi、Meme 或风险筛选。
  2. 查看是否有巨鲸、交易所流入或解锁窗口。
  3. 把高价值资产放进报告库或日历观察。

Daily 简报流程

每日情报压缩

每天只保留少量高价值信息：核心判断、巨鲸动作、风险事件、热门板块和明日观察节点。

  1. 从今日情报查看最重要的 5 条。
  2. 把风险事件放入日历和预警中心。
  3. 把需要追踪的钱包或资产沉淀为报告。

Operating Rules

调查原则

好的情报流程要能降低误读，而不是把每个链上动作都变成交易冲动。

先事实

先确认交易、地址、金额和时间，再讨论可能含义。

再标签

实体标签可以辅助解释，但敏感标签必须看来源和置信度。

后推断

资金意图、交易员风格和板块热度都属于推断，需要标注边界。

持续观察

重要线索应进入报告、日历或预警，而不是立即当作结论。
