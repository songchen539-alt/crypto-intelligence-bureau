Roadmap

产品路线图

先把安全浏览和核心产品结构做好，再逐步接入真实链上数据、账号系统、提醒渠道和团队 API。

路线原则

每一步都围绕用户为什么来：查钱包、查巨鲸、查交易员、看核心情报。

Milestones

开发阶段

路线图会随着数据源和产品验证结果持续调整。

Phase 1

公开官网与核心页面

首页、今日情报、钱包调查、巨鲸雷达、交易员调查、钱包监控、Crypto MBTI。

Phase 2

真实链上数据接入

接入钱包余额、交易历史、Token 持仓、交易所流入流出和巨鲸资金流。

Phase 3

账号与监控提醒

支持关注列表、监控规则、Email / Telegram / Discord / Webhook 提醒。

Phase 4

团队与 API

开放团队协作、批量钱包调查、API 密钥、速率限制和数据导出。
